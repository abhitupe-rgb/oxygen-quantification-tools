⚠ Model assumptions — replace with real data before use in the field

  • PSA plant CAPEX: Flat input figure. Real plants scale non-linearly by capacity — get vendor quotes per tier.
  • Pipeline cost ($150/m default): Highly variable. Most important source of uncertainty in the model.
  • O₂ demand per bed: Using 35% average utilisation. Validate against actual facility data.
  • Hub downtime risk: Not modelled. A single hub failure affects all spokes. Consider a cylinder backup cost at each spoke.
  • Demand aggregation: Hub sized for aggregate average + buffer. Simultaneous surge across all spokes is less likely than individual peaks.
